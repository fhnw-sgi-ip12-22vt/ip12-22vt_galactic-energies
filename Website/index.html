Startseite

Digitaler Alltag

Zusammenfassung

Erlebnisreiches Arcade-Spiel mit Kurbel und Joystick im Weltall.

Schlüsselbegriffe

Sprachen: Java 17, JavaFX
Datenbank: MariaDB 10.5
Betriebssystem: Raspberry Pi OS based on Debian Version: 11
Hardware: Raspberry Pi 4, Joystick, Kurbel, Button, NA219 Strommesser, LED-Lämpchen
Dokumentation: Confluence, GitLab

Zielsetzung

Das Strombewusstsein soll durch Galactic Energies gefördert werden. Hierzu kann durch Muskelkraft Strom produziert werden. Die dadurch erzeugte Energie soll in Bezug zu dem Energieverbrauch von Alltagsgegenständen gesetzt werden.

Ausgangslage

Aufgrund der Unwissenheit von den Jugendlichen zwischen dem Alter von 12 und 16 Jahren, wurde dieses Arcade-Spiel entwickelt, um eine Wissensvermittlung über den Stromverbrauch von alltäglichen Gegenständen verwirklichen zu können.
